# Infinite Scroll

#### React Contact List to demonstrate infinite scrolling feature

Project in Action - [Infinite Scroll](https://infinite-contacts.netlify.app/)

## Usage

### Install Dependencies (frontend)

```
npm install
```

### Run

```
# Run frontend (:3000)
npm start

```

## Build & Deploy

```
# Create frontend prod build
npm run build

```
